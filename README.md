2. Ekstrak file ZIP ke lokasi yang diinginkan.  
3. Buka **Command Prompt (CMD)**.  
4. Arahkan CMD ke folder hasil ekstraksi ZIP. Contoh:  
   ```bash
   cd C:\Users\LENOVO\Downloads\tugas-akhir-kriptografi-main
